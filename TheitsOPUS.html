VBR / CVBR on Mobile Phone Networks

For live 4G and 5G links
Setting Mobile phone network implication Recommendation
VBR Off Most predictable packet sizes, but less efficient audio quality Use only if fixed packet size is required
VBR On, CVBR Off Can produce larger packet spikes, which may worsen jitter on 4G/5G Avoid for live mobile links
VBR On, CVBR On Good quality while limiting packet size spikes Recommended
VBR Off, CVBR On Same practical result as VBR Off because CVBR only matters with VBR Not useful
